# [게임 맵 최단거리](https://school.programmers.co.kr/learn/courses/30/lessons/1844)

### Solution

매우 기초적인 BFS<br>
만약 DFS로 푼다면 visit의 숫자(방문까지 걸리는 이동횟수)를 비교해서 낮게 탐색할 수 있는지 확인해야함

```cpp
int solution(vector<vector<int>> maps)
{
    int answer = -1;
    int n = maps.size();
    int m = maps[0].size();
    
    queue<pair<int,int>> q;
    vector<vector<int>> visit(n, vector<int>(m, 0));
    
    int dx[4] = {0,1,0,-1};
    int dy[4] = {1,0,-1,0};
    q.push({0,0});
    visit[0][0] = 1;
    while(!q.empty()){
        int cx = q.front().second;
        int cy = q.front().first;
        //탈출 조건
        if(cx == m-1 && cy == n-1) 
            return visit[cy][cx];
        
        q.pop();
        for(int i = 0 ; i < 4 ; i++)
        {
            int nx = cx+dx[i];
            int ny = cy+dy[i];
            //다음 위치에 갈 수 있는지 판단
            if(nx >= 0 && ny >= 0 && nx < m && ny < n){
                if( maps[ny][nx] == 1 && visit[ny][nx] == 0 ){
                    q.push({ny,nx});
                    visit[ny][nx] = visit[cy][cx]+1;    
                }
            }
        }
    }
    
    //도착 불가능
    return -1;
}
```
